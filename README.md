# Music Playlist

## Project Description

This is a playlist built with HTML, CSS and JavaScript.

## Functionality

- Display playlists from a JavaScript array
- Click a playlist to see genres, artists and songs
- If a playlist has no genres, No genres yet is shown
- Add new playlists using a form

## Design

The layout is built with Flexbox and has red background

## Project Structure

music-playlist/
├── index.html
├── style.css
└── script.js

## How to Run

1. Clone or download the repository
2. Open index.html in your browser
